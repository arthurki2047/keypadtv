# **App Name**: Keypad TV

## Core Features:

- Channel Listing: Display available live TV channels from the provided URL (https://allinonereborn.xyz/livetv-hub/).
- Live Stream: Play the selected live TV stream.
- Keypad Navigation: Enable navigation using keypad input.
- Search Functionality: Add a search function with a UI optimized for keypad use.

## Style Guidelines:

- Primary color: Deep purple (#4A148C) to convey a sense of reliability.
- Background color: Very light purple (#F3E5F5). Almost white, for high readability and a clean look.
- Accent color: Electric green (#7CFC00) for clear visual cues.
- Body and headline font: 'PT Sans', sans-serif, for maximum readability.
- Use large, clear, and easily recognizable icons for navigation and channel selection.
- Simple, grid-based layout optimized for keypad navigation with large, touch-friendly elements.
- Smooth transitions between channels and screens, but keep them subtle for keypad phone performance.
